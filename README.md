## 📌 Descrizione

**OfficinaIO** è un'applicazione Java da linea di comando pensata per simulare la gestione di una piccola officina.

Il progetto permette di rappresentare in modo semplice alcune operazioni tipiche, come la gestione delle prenotazioni, degli interventi e dei costi associati ai servizi dell'officina.

## 🛠️ Compilazione

Clona la repository:

```bash
git clone https://github.com/LoRy24/OfficinaIO.git
cd OfficinaIO
```

Compila il progetto con Gradle:

```bash
./gradlew build
```

Su Windows:

```bash
gradlew.bat build
```

## ⚙️ Note sul funzionamento

L'applicazione è strutturata come simulatore CLI e organizza la logica dell'officina tramite classi Java dedicate.

Il programma gestisce dati e operazioni legate al contesto di un'officina, mantenendo una struttura semplice, leggibile e facilmente estendibile.

Alcune funzionalità possono essere ancora in fase di sviluppo o miglioramento.
